# 图书简介开发记录

## 适用范围
- 功能：图书简介字段、编辑表单、批量导入、简介数据回填。
- 相关文件：
  - `backend/src/routes/books.ts`
  - `backend/src/routes/books-import.ts`
  - `backend/src/scripts/backfill-book-introductions.ts`
  - `vue-vben-admin/apps/web-ele/src/views/library/*`

## 当前状态
- 图书已支持 `introduction` 字段
- 简介只在新增/编辑图书抽屉中展示与编辑，不在列表表格展示
- Excel 批量导入已支持简介列
- 已提供简介回填脚本

## 目录
- `2026-03-07 v1.0`：图书简介字段接入
- `2026-03-07 v1.1`：简介数据回填脚本

## 2026-03-07 v1.0

### 范围
- 在“图书管理”新增 `introduction`
- 列表表格不展示简介
- 新增/编辑抽屉支持简介
- Excel 导入支持简介列

### 不做
- 不增加简介搜索
- 不在借阅流程的图书查找接口中使用简介

### 数据与校验
- `introduction`：可选字符串，去除首尾空格
- 后端限制：`<= 300` 字符

### 后端改动点
- `BookDoc` 增加 `introduction?: string`
- 接口：
  - `GET /api/books` 返回 `introduction`
  - `POST /api/books` 支持写入
  - `PUT /api/books/:isbn` 支持更新

### 批量导入
- Excel 表头支持：
  - `introduction`
  - `简介`
- 导入策略：
  - 新书：写入简介
  - `increment_stock`：不改简介
  - `skip`：不改简介
  - `overwrite`：当 Excel 有值时更新简介

### 前端改动点
- 新增/编辑抽屉增加 `简介` 字段
  - `textarea`
  - `maxlength: 300`
  - `showWordLimit: true`
  - `autosize`
- 列表表格不新增简介列
- 导入预览表格不展示简介列

### 数据来源说明
- 系统不自动联网抓取简介
- 简介由管理员手动录入或 Excel 导入
- 演示数据建议自行改写，避免直接复制公开原文

### 验收点
- 新增图书可输入并回填简介
- 编辑图书可修改简介
- 列表表格不出现简介列
- 批量导入能按策略写入或更新简介

### 冒烟记录
- 自检：
  - `pnpm -C backend build` 通过
  - `pnpm -C vue-vben-admin -F @vben/web-ele typecheck` 通过
- API 冒烟：
  - `POST /api/auth/login` 成功
  - `GET /api/books` 返回字段包含 `introduction`
  - `PUT /api/books/:isbn` 提交 301 字简介返回 `400`
- 备注：
  - 测试 ISBN `9900000000000` 已从 MongoDB 删除

## 2026-03-07 v1.1

### 目标
- 为 MongoDB 中“简介为空”的图书批量补齐 `introduction`
- 测试数据保持为空
- 不覆盖已有简介

### 数据来源与规则
- 优先从 Open Library 按 ISBN 获取公开描述
- 缺失时用本地模板生成简短简介
- 最终写入长度强制截断到 `<= 300`
- 跳过明显测试数据：
  - `测试`
  - `test`
  - `demo`
  - `mock`
  - `前导0`
  - `占位`

### 实现改动点
- 新增脚本：`backend/src/scripts/backfill-book-introductions.ts`
- 支持：
  - `--dry-run`
  - `--limit`
- 默认只更新 `introduction` 为空或不存在的记录

### 验收点
- 非测试数据的图书简介基本补齐
- 测试数据简介仍为空
- 已有简介不被覆盖

  - `pnpm -C backend backfill:book-intros -- --dry-run --limit 12`
- 实际回填：
  - `pnpm -C backend backfill:book-intros`
- 结果：

  - 修正关键词误判后追加更新 `1` 条
